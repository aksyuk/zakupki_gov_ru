
## Волшебные сокращения    

* **ЗК** -- запрос котировок    
* **ЗК-БИ** -- запрос котировок без извещения    
* **ЗП** -- запрос котировок     
* **ОК** -- открытый конкурс    
* **ОК-Д** -- открытый конкурс двухэтапный    
* **ОК-ОУ** -- открытый конкурс с ограниченным участием    
* **ПО** -- предварительный отбор    
* **ЭА** -- электронный аукцион
